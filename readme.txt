## Description

Dream Coming Soon & Maintenance is a simple plugin that allows you to enable or disable a "Coming Soon" mode on your WordPress site. When enabled, it redirects visitors to a static coming soon page.

## Features

- Easily enable or disable "Coming Soon" mode.
- Redirects visitors to a static coming soon page.
- Customizable static coming soon page.

## Installation

1. Upload the `dream-coming-soon` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Settings' > 'Coming Soon' to configure your coming soon settings.

## Frequently Asked Questions

### How do I enable Coming Soon mode?

1. Go to your WordPress dashboard.
2. Navigate to 'Settings' > 'Coming Soon'
3. Enable the "Coming Soon" mode and save your changes.

### How do I customize the static coming soon page?

The static coming soon page is a standard HTML file located in the plugin directory. You can customize it by editing the `coming-soon.php` file.
